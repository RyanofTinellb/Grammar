Glossing Rules

The Leipzig glossing rules are used throughout this site. A summary thereof will be presented here. The three letter abbreviations used for functional morphemes are given on the next page.

  1. A content morpheme is written in either roman or italic type.
  2. A function morpheme is written in small-caps.
  3. A hyphen ( - ) is used to separate morphemes.
  4. When two or more English content morphemes correspond to a single Tinellbian morpheme, an underscore ( _ ) is used to separate them.
  5. When a single Tinellbian function morpheme fulfills two or more grammatical roles, a semi-colon ( ; ) is used to separate them.
