Gerunds

The gerund suffix -ru w changes basic verbs into nouns describing the activity.

hubaru Vbw breathing

malaru mlw reflecting

mikuru M>w being red

Compound verbs, as well as those involving noun incorporation, can also be changed like this.

dissu’ahatiru D;zahTw feeling happy

pakicuru pKyw believing to be true

’i’issiru ii;Sw singing

The genitive determiners, including -qa q, allow for (only) the subject of the verb to be mentioned.

This continues to apply in instances using argument promotion.
